Research Publications

Temporal Analysis and CWE Code Prediction for Software Vulnerabilities using Machine Learning

8th IEEE International Conference on Computational Systems & Information Technology for Sustainable Solutions (CSITSS) Nov 2024 Paper ID: 943

Research focusing on temporal analysis of software vulnerabilities and using machine learning for CWE code prediction. Predicts potential zero-day attacks through temporal analysis of historical CWE data, uncovering patterns to implement proactive security measures. The predictive model achieved balanced performance with an accuracy of 0.66, precision of 0.65, F1 score of 0.64, and recall of 0.66.

Machine Learning Security CWE

Leveraging XGBoost Machine Learning Algorithm for Common Vulnerabilities and Exposures Exploitability Classification

8th IEEE International Conference on Computational Systems & Information Technology for Sustainable Solutions (CSITSS) Nov 2024 Paper ID: 611

Implementation of XGBoost algorithm for classifying CVE exploitability, enhancing vulnerability assessment processes. The model achieved an accuracy of 94%, precision of 92%, recall of 91%, and F1-score of 92%, proving efficient in distinguishing between exploitable and non-exploitable vulnerabilities.

XGBoost CVE Classification

Data Breaches: Financial and Reputational Impacts of Vulnerabilities on Organizations to Enhance Cybersecurity Strategies

3rd International Conference on Automation, Computing and Renewable Systems Dec 2024 Upcoming

Analysing the impact of cyberattacks on organizations. Examines and monitors vulnerabilities within organizations using Power BI to identify trends in data breaches, highlighting the primary methods of breaches and offering insights to enhance cybersecurity strategies.

Vendor and Product Vulnerability Trends: An Analysis of the CVE Database

3rd International Conference on Security and Privacy (ICSP 2024) Nov 2024

A detailed analysis of vulnerabilities associated with vendors using data from the Common Vulnerabilities and Exposures (CVE) database, focusing on top vendors and products identified through an in-depth analysis spanning from 2013 to 2023.
